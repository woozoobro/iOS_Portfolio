# Tuple이란?

## Tuple 그게 뭐야!
```swift
let a = ("우주형", 28, "편집")
```
이렇게 괄호 안에 원하는 타입을 나열해서 쓰는 게 튜플
타입을 여러가지 막 써도 된다

```swift
a.0
a.1
a.2
```
![](https://velog.velcdn.com/images/woojusm/post/ff6185ac-112b-4572-9c16-76223409eeb8/image.png)
요렇게 튜플 값에 접근할 때는 순서대로 0, 1, 2 로 접근할 수 있음
